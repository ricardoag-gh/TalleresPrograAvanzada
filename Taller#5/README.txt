Instrucciones para correr el programa :

El programa se corre por medio del el archivo de python "main.py", es una aplicación que corre solamente en consola. Para escritura y lectura se utilizó solamente el json file "existing_users.json".


Descripción del proyecto :

El proyecto cumple con lo solicitado en el enunciado :

1- Validar usuario y contraseña de usuarios existentes en un login usando datos del json file.
2- Permite registrar nuevos usuario en el json file con las validaciones solicitadas en usuario y contraseñas.
3- Listar usuarios existentes del json con sus respectivas propiedades.
